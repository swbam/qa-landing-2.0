How to Use

Add the following code to your website to embed the QuickAuction component:

In the <head> section:

<!-- React and ReactDOM -->
<script src="https://unpkg.com/react@18.3.1/umd/react.production.min.js"></script>
<script src="https://unpkg.com/react-dom@18.3.1/umd/react-dom.production.min.js"></script>

<!-- CSS Variables -->
<style>
:root {
  --primary: #0054da;
  --primary-dark: #0049bb;
}
</style>

<!-- Component CSS -->
<link rel="stylesheet" href="https://qa-component.vercel.app/quickauction-search.css">

Where you want the component to appear:

<div class="quickauction-search-component" data-primary-color="#0054da" data-primary-dark-color="#0049bb"></div>

Before the closing </body> tag:

<script src="https://qa-component.vercel.app/quickauction-search.js"></script>

Note: Replace "qa-component.vercel.app" with your actual Vercel deployment URL.

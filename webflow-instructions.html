QuickAuction Webflow Component - Implementation Guide

1. Add to Head Tag

Copy and paste this code into the "Custom Code" section under the "Before </head> tag" in your Webflow page settings:

<!-- React and ReactDOM -->
<script src="https://unpkg.com/react@18.3.1/umd/react.production.min.js"></script>
<script src="https://unpkg.com/react-dom@18.3.1/umd/react-dom.production.min.js"></script>

<!-- CSS Variables -->
<style>
:root {
  --primary: #0054da;
  --primary-dark: #0049bb;
}
</style>

<!-- Component CSS -->
<link rel="stylesheet" href="https://YOUR_S3_BUCKET.s3.YOUR_REGION.amazonaws.com/quickauction-search.css">

2. Add to Body Tag

Copy and paste this code into the "Custom Code" section under the "Before </body> tag" in your Webflow page settings:

<!-- Component JS -->
<script src="https://YOUR_S3_BUCKET.s3.YOUR_REGION.amazonaws.com/quickauction-search.js"></script>

3. Add Component to Page

Add a div element to your page with the class "quickauction-search-component". You can do this by:

  1. Adding a new div element in the Webflow designer
  2. Setting its class to "quickauction-search-component" in the Class field
  3. Setting its ID to something memorable like "qa-search"

Or you can add an HTML embed element and paste this code:

<div class="quickauction-search-component" data-primary-color="#0054da" data-primary-dark-color="#0049bb"></div>

4. Customize Component Colors (Optional)

You can customize the colors by changing the data attributes:

<div class="quickauction-search-component" 
     data-primary-color="#FF5722" 
     data-primary-dark-color="#E64A19">
</div>

And update the CSS variables in the head section to match:

<style>
:root {
  --primary: #FF5722;
  --primary-dark: #E64A19;
}
</style>

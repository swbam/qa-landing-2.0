How to Embed the QuickAuction Component

Integration Instructions

Follow these steps to add the QuickAuction component to your existing Webflow site:

1. Add these tags to your site's <head> section:

<!-- React and ReactDOM -->
<script src="https://unpkg.com/react@18.3.1/umd/react.production.min.js"></script>
<script src="https://unpkg.com/react-dom@18.3.1/umd/react-dom.production.min.js"></script>

<!-- CSS Variables -->
<style>
:root {
  --primary: #0054da;
  --primary-dark: #0049bb;
}
</style>

<!-- Component CSS -->
<link rel="stylesheet" href="https://qa-landing-2.vercel.app/quickauction-search.css">

2. Add this div where you want the component to appear:

<div class="quickauction-search-component" data-primary-color="#0054da" data-primary-dark-color="#0049bb"></div>

3. Add this script just before the closing </body> tag:

<script src="https://qa-landing-2.vercel.app/quickauction-search.js"></script>
